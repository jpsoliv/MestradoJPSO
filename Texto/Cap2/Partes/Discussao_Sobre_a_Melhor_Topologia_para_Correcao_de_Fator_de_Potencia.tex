\section{Discussão sobre a melhor topologia para correção de fator de potência}

Para a operação segura e de acordo com o cumprimento dos requisitos aeronáuticos, são necessários procedimentos para suprimir o alto \textit{THD} nos sistema elétrico de uma aeronave, de modo que seus efeitos não sejam prejudiciais aos equipamentos instalados na rede. 
Dentre os diversos métodos de mitigação de harmônicas nas redes do sistema elétrico de uma aeronave, destacam-se diversas topologias onde cada uma possui suas vantagens, desvantagens e peculiaridades. Nesse trabalho será descrito duas classes de sistemas de controle de harmônicas: os passivos e os ativos.

\subsection{Sistemas Passivos} 

A caracterização de um sistema passivo dá-se pela ausência de um controle ativo para o controle de comutação de dispositivos semicondutores. Neste tipo de sistema, na presença de semicondutores, a comutação é desempenhada por diodos, as quais não necessitam de comando de um controle externo para a que haja a abertura ou o bloqueio do fluxo de corrente entre seus terminais. Outro sistema passivo é dado por filtros casados com as harmônicas de alta frequências do sistema.
